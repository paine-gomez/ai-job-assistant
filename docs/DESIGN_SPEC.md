# 设计规范文档

## 设计理念

**专业、简洁、现代**——符合 AI 产品审美。
深色为主调，体现技术感和专业度，卡片式布局，信息层次分明。

---

## 配色方案

| 用途 | 色值 | 说明 |
|------|------|------|
| 主背景 | `#0a0a0b` | 深黑，护眼且有质感 |
| 卡片背景 | `#1a1a1f` | 深灰，与背景形成层次 |
| 主色调 | `#6366f1` (Indigo) | 品牌色，按钮、链接、高亮 |
| 辅色调 | `#22d3ee` (Cyan) | 标签、进度条、强调 |
| 成功色 | `#34d399` (Green) | 匹配度高、通过状态 |
| 警告色 | `#fbbf24` (Yellow) | 匹配度中、需注意 |
| 危险色 | `#f87171` (Red) | 不匹配、错误 |
| 文字主色 | `#f1f1f3` | 近白，主文字 |
| 文字次色 | `#a1a1aa` | 灰色，辅助文字 |
| 边框色 | `#2a2a30` | 分割线、卡片边框 |

## 组件规范

### 导航栏 (Navbar)
- 固定在顶部，高度 56px
- 左侧 Logo + 项目名，右侧导航链接
- 背景半透明 + 毛玻璃效果 (`backdrop-blur`)
- 底部 1px 边框分割

### 功能卡片 (FeatureCard)
- 深灰背景圆角卡片 (`rounded-xl`)
- 悬停时边框变为主色调，有轻微上浮效果
- 图标（emoji）+ 标题 + 简介 + 跳转按钮

### 按钮 (Button)
- 主按钮：Indigo 背景 + 白色文字
- 次按钮：透明背景 + 边框
- 危险操作（删除）：红色
- 大小：默认 `h-10 px-4`，图标按钮 `h-9 w-9`

### 输入框 (Input / Textarea)
- 深灰背景 + 浅色边框
- 聚焦时边框变 Indigo
- placeholder 文字用灰色

### 消息气泡 (MessageBubble)
- 用户消息：右对齐，Indigo 背景
- AI 消息：左对齐，深灰背景
- AI 回复支持 Markdown 渲染（标题、列表、加粗）
- 流式输出时文字逐字出现

### 评分展示 (ScoreDisplay)
- 总评分用大号数字（如 "72分"）突出显示
- 各维度用 shadcn/ui Progress 横向进度条
- 颜色根据分数变化：绿(≥80) / 黄(60-79) / 红(<60)
- MVP 不做环形进度图，用简单直观的进度条替代

## 响应式适配

本次 MVP 优先适配桌面端（≥1024px），移动端后续优化。
基础断点：
- Desktop: ≥1024px（主要适配目标）
- Tablet: 768-1023px（左栏收起）
- Mobile: <768px（暂不深度适配，保证基本可用即可）

---

## 交互细节

- **加载状态：** AI 思考时显示跳动的三个点动画
- **流式输出：** 文字逐字显示，光标闪烁
- **错误提示：** 顶部弹出 Toast，3秒后自动消失，红色背景
- **成功提示：** 上传成功/删除成功后 Toast，绿色背景
- **空状态：** 无文档时显示引导图示 + "上传你的第一份资料"
- **确认操作：** 删除文档前弹出确认框
